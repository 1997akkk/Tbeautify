# Tbeautify-for-Typecho
十大特效美化插件，简称：Tbeautify。
包含春节灯笼，网站置灰，鼠标点击效果，悬浮音乐播放器，看板娘等诸多特效！后续会不断新增更多特效~敬请关注
github开源地址：[https://github.com/1997akkk/Tbeautify][1]

### 使用方法

1. 下载本插件，放在 usr/plugins/ 目录中，文件名命名为Tbeautify
2. 登录管理后台，激活插件
3. 开启你所想要的网站特效
